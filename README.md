### Try it now
Go to https://canvasdraw.limboy.me to start drawing.

### Installation
#### Requirements
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

#### Clone the repo

```
git clone https://github.com/limboy/canvasdraw.git
```

#### Install the dependencies

```
pnpm install
```

#### Start the server

```
pnpm build
pnpm dev
```

open http://localhost:3727 and start drawing.
